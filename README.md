<h1>Rabbit</h1>

An IoT device with RGB led controller built into my kids' rabbit light.

<h2>Sensors</h2>
<ul>
  <li>BME280 temperature/humidity/pressure</li>
  <li>RCWL-0516 presence</li>
  <li>LDR light</li>
</ul>

<h2>Dependencies</h2>

</ul>
